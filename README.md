## Setup / Instructions:

**Setting up dependencies:**
1. If you don't have it already, install npm. (https://www.npmjs.com/)
2. Clone repository.
3. Through the terminal, type:

   ```npm install```

**Starting server:**
1. In your clone repository (with npm installed), type:

   ```cd jzstyles``` <- (Make sure that you are in the /jzstyles directory, not the outside repository folder.)
   
2. Next, type:
   
   ```npm run dev```
   
3. You will see a success message in your console logs.
4. Through your browser, you can now access the site through: http://localhost:3000
