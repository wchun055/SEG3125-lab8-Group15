## Setup / Instructions:

### Setting up dependencies:
1. If you don't have it already, install [npm](https://www.npmjs.com/).
2. Clone repository.
3. In your clone repository type:

   ```cd jzstyles``` <- **(IMPORTANT! Make sure that you are in the /jzstyles directory.)**
   
4. In the terminal, type:

   ```npm install```

5. [daisyUI 5](https://www.npmjs.com/package/daisyui) is also a requirement. In the terminal, type:

   ```npm i -D daisyui@latest```

### Starting server:
1. In your clone repository (with npm installed), type:

   ```cd jzstyles``` <- **(IMPORTANT! Make sure that you are in the /jzstyles directory.)**
   
2. Next, type:
   
   ```npm run dev```
       
3. You will see a success message in your console logs.
4. Through your browser, you can now access the site landing page through: http://localhost:3000
